Visão Geral do atendimento.ao

O atendimento.ao é uma plataforma de atendimento ao cliente omnichannel desenvolvida especificamente para pequenas e médias empresas angolanas. A solução centraliza todos os canais de comunicação (WhatsApp, Facebook, Instagram, e-mail, telefone) em uma única interface, facilitando o gerenciamento e melhorando a experiência do cliente.

Problema que resolve: Muitas PMEs não têm canais centralizados para atender clientes, com comunicação dispersa across múltiplas plataformas.

Diferenciais no mercado angolano:

  • Domínio nacional (.ao) e presença local
  • Suporte em português com conhecimento do mercado angolano
  • Preços em Kwanza (Kz) acessíveis para PMEs
  • Integração com ferramentas e sistemas locais
  • Gateway de pagamento local (EMIS, Multicaixa Express)

Informações Técnicas

Arquitetura da Plataforma:

Núcleo: Chatwoot (plataforma de atendimento omnichannel open-source)

Hospedagem: Self-hosted em cluster Docker Swarm com Traefik

Modelo: Multi-tenant (cada cliente isolado no mesmo servidor)

Ferramentas de Automação e Integração:

  • n8n: Automação de workflows e integrações
  • Botpress: Criação de chatbots para FAQ
  • Metabase: Business Intelligence e relatórios
  • Kill Bill: Sistema de billing open-source
  • Keycloak: Gestão de autenticação e usuários

Integrações de Canais:

  • WhatsApp Business API
  • Facebook Messenger
  • Instagram Direct
  • E-mail (SMTP/IMAP)
  • Telegram
  • Widget para site

Integrações Locais (Angola):

  • Gateway de pagamento (EMIS, Multicaixa Express)
  • Sistema de faturação (Conectados.ao)
  • APIs de operadoras de telecomunicações

Funcionalidades Principais

Atendimento Omnichannel:

  • Centralização de todos os canais em uma única plataforma
  • Visão unificada das conversas com cada cliente
  • Distribuição automática de conversas para atendentes

Automação:

  • Chatbots para perguntas frequentes (FAQ)
  • Respostas automáticas e mensagens de saudação
  • Encaminhamento automático por palavras-chave
  • Workflows automatizados (ex: criar ticket no CRM)

Relatórios e Analytics:

  • Métricas de desempenho (tempo de resposta, satisfação)
  • Relatórios personalizados por período
  • Dashboard com indicadores-chave (KPI)
  • Exportação de dados para análise

Gestão de Clientes:

  • Base de dados de clientes unificada
  • Histórico completo de interações
  • Tags e segmentação de clientes
  • Integração com CRM
